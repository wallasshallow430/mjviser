## 🖥️ What is mjviser?

mjviser is a web-based MuJoCo viewer powered by Viser. It lets you open and inspect 3D scenes in a browser on Windows. It is useful for robotics work, simulation checks, and quick scene review without setting up a heavy desktop tool.

Use it to:
- View MuJoCo models in a browser
- Inspect robot poses and scene layout
- Check visuals for simulation work
- Share a view with other people on the same network

## 🔧 How It Works

mjviser starts a local viewer on your computer. You then open it in a browser and use the page to inspect the scene.

Common parts of the app:
- 3D view area for the model
- Camera controls for rotate, pan, and zoom
- Scene loading for MuJoCo files
- Browser-based display for easy access

This setup keeps the workflow simple. You do not need to learn a complex desktop interface to view simulation scenes.

## 🎮 Basic Controls

These controls are common in 3D viewers and should feel familiar:

- Left mouse button: rotate the view
- Right mouse button: pan across the scene
- Mouse wheel: zoom in and out
- Drag over the view: move the camera
- Reset view: return to the default camera angle if needed

If the scene looks too close or too far away, use the mouse wheel to adjust it.
